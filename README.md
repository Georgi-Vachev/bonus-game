# JS Bonus Chest Game

A simple chest-opening game built with **Pixi.js** and **GSAP**.  

Tested on **Node.js v20.20.1

## Features

- Play button to start the game  
- 6 interactive chests  
- Random outcomes: lose, win, bonus  
- Animated chest opening using GSAP  
- Total win counter
- Bonus screen with separate animation  

---

## Installation

```bash
git clone <https://github.com/Georgi-Vachev/bonus-game>
cd <bonus-game>
npm install
```

## Running the project

1. For preview

```bash
npm run preview
```

2. For development

```bash
npm run dev
```

## Build for production

```bash
npm run build
```
